Elgato / Corsair  ·  2021–2022

Design
System

This project was kicked off, when the Elgato team started planning to move their website to a different CMS System. A component based Design System helped all involved parties to make the platform shift as efficient as possible.

Role
Lead UI Designer
Type
Design System, Component Library
Period
2021–2022
Why a Design System

The design system was built in Figma and follows the atomic design methodology, whereas all final page components can be broken down into multiple distinctive stages. Multiple variants were built within each component to allow an easy usage of elements with slightly different properties.

The Structure

Atoms Colors, Typography, Icons, Logos, Toggles and much more are building the foundation of the overall design system.

Molecules Components like Buttons, Text Styles, Popups and Drop-downs make use of existing Atoms to help create more complex Organisms.

Organisms Module content like gallery images, comparison tables, product tiles and grid layouts was grouped under Organisms and includes all content that is needed to create full panel templates.

Templates With the help of fully built out panel modules, the development team can create a toolbox of reusable templates in the CMS system. This enables the Elgato team to easily build up new pages from existing modules.
